Red, The Dragon

Ludum Dare Competition

Role: Everything

2011


Red, The Dragon was developed in Microsoft XNA by myself for a Ludum Dare competition themed on "Escape". The player controls a green dragon named Red to evade or shoot down the enemy projectiles, eventually fighting a "final boss".

VacaMaia

Vaca Vitória Software

Role: Programmer, Game Designer

2013


VacaMaia is a short "Time Challenge" game, developed in Unity3D by Vaca Vitória Software for a local event. The player is supposed to collect 10 coins before time runs out.

Virtual Vision

Bit Temple Games, Global Game Jam 2014

Role: Programmer

2014


Virtual Vision was developed in Unity3D by Bit Temple Games for the Global Game Jam 2014 competition. It is a scifi platformer puzzle game involving gravity, anti-gravity and removal of obstacles.
